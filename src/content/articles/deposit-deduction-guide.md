# 退去費用の相場と交渉術｜原状回復で損しない方法

「敷金30万円預けていたのに、退去時に追加で20万円請求された」「壁紙の全面張替えを請求されたけど、これって本当に自分が払うの？」——賃貸退去でこんな経験、もしくは不安を抱えている方は少なくありません。

結論から言うと、**退去費用の過剰請求は今も頻発しています**。しかし同時に、**借主の権利を正しく理解していれば、ほとんどのケースで大幅に減額できます**。その根拠は国土交通省が公表している「原状回復をめぐるトラブルとガイドライン」です。

筆者もかつて、退去時に「ハウスクリーニング10万円＋壁紙全面張替え15万円」と見積もられ、ガイドラインを根拠に交渉して**最終的に3万円の実費負担**に抑えた経験があります。この記事では、その時に使った知識と、賃貸実務に携わる専門家の解説をもとに、**2026年時点で通用する退去費用の相場と交渉術**をまとめました。

読み終わる頃には、「何を払うべきで、何を拒否していいか」がはっきりと分かるはずです。

## 国土交通省「原状回復ガイドライン」の考え方

退去費用の交渉で最強の武器になるのが、国土交通省が公表している**「原状回復をめぐるトラブルとガイドライン」** です。裁判の判例を集約して作られており、家賃滞納や過剰請求をめぐる訴訟でも参照される、いわば業界の共通ルールブックです。

### ガイドラインの核心｜原状回復とは「元通り」ではない

多くの人が誤解しているのは、**「原状回復＝入居時と全く同じ状態に戻す」** という考え方です。ガイドラインでは、原状回復を次のように定義しています。

> 賃借人の居住、使用により発生した建物価値の減少のうち、**賃借人の故意・過失、善管注意義務違反、その他通常の使用を超えるような使用による損耗・毀損** を復旧すること

つまり、**「普通に住んでいればついてしまう汚れや傷」は貸主負担**、**「普通ではない使い方による損傷」だけが借主負担**、というのが大原則です。

### 借主負担と貸主負担の区分

| 区分 | 具体例 | 負担者 |
|------|-------|--------|
| 経年劣化 | 日焼けによる壁紙の変色、畳の色あせ | **貸主** |
| 通常損耗 | 家具の設置跡、冷蔵庫背面の黒ずみ | **貸主** |
| 故意・過失 | タバコのヤニ汚れ、ペットの引っかき傷 | **借主** |
| 善管注意義務違反 | 結露の放置によるカビ、水漏れ放置 | **借主** |
| 通常を超える使用 | クギを刺した下地ボードまでの穴 | **借主** |

これを押さえるだけで、請求書の9割は反論できるようになります。

ガイドラインは国交省のサイトから無料でPDF閲覧できます。一度目を通しておくと、退去時の交渉力が劇的に上がります。

## 敷金から差し引かれる項目の相場

では実際、退去時にはどんな項目がいくらで請求されるのか。**全国平均の相場**を整理しました。

### 主要項目の請求相場（ワンルーム〜1LDKの場合）

| 項目 | 相場（目安） | 借主負担になるケース |
|------|-------------|---------------------|
| ハウスクリーニング | 20,000〜50,000円 | 特約があれば借主負担（後述） |
| エアコンクリーニング | 8,000〜15,000円/台 | 通常は貸主負担、ヤニ汚れは借主 |
| 壁紙（クロス）張替え | 1,000〜1,500円/㎡ | タバコ・ペット・故意の汚損のみ |
| フローリング補修 | 部分20,000〜50,000円、全面80,000〜200,000円 | 深い傷・水漏れ跡など |
| 畳の表替え | 1枚4,000〜8,000円 | 通常は経年扱いで貸主負担 |
| 鍵交換 | 10,000〜20,000円 | 特約があれば借主負担 |
| 浴室・水回りクリーニング | 15,000〜30,000円 | カビ放置なら借主 |

### ファミリー・2LDK以上の相場感

3LDK以上だと総額10〜30万円台の請求も珍しくありません。ここに**過剰な「全面張替え」請求**が入ると、一気に40〜60万円に膨らみます。ただし後述するように、**経年劣化分は貸主負担**のため、大半は減額可能です。

### 減価償却の考え方（ここが一番重要）

壁紙やフローリングには**「耐用年数」** が設定されており、時間が経つほど借主負担割合は下がります。

| 設備 | 耐用年数 | 6年住んだ場合の借主負担 |
|------|---------|------------------------|
| クロス（壁紙） | 6年 | ほぼゼロ（残存価値1円扱い） |
| カーペット | 6年 | ほぼゼロ |
| クッションフロア | 6年 | ほぼゼロ |
| フローリング | 建物の耐用年数（木造22年等） | 長期 |
| 畳表 | 6年 | ほぼゼロ |
| エアコン | 6年 | ほぼゼロ |

**6年以上住んだ部屋の壁紙張替えは、原則全額借主負担にはならない**。これを知らずに「60㎡分×1,200円＝72,000円請求」を丸呑みしてしまう人が非常に多いのです。

## ハウスクリーニング費用の相場と「特約」の落とし穴

退去時請求で最も多いトラブルが、**ハウスクリーニング費用**です。

### 間取り別の相場

| 間取り | 相場 |
|--------|------|
| ワンルーム・1K | 15,000〜30,000円 |
| 1LDK・2K | 25,000〜45,000円 |
| 2LDK・3K | 35,000〜60,000円 |
| 3LDK以上 | 50,000〜100,000円 |

### 特約があれば借主負担、ただし…

契約書に「退去時のハウスクリーニング費用は借主負担」と明記されていれば、ガイドラインに反していても**基本的には有効**とされています（判例あり）。

ただし、有効となるには次の条件を満たす必要があります。

1. 契約書に**具体的な金額（またはその算出根拠）** が明記されている
2. 借主が**その内容を理解して契約した**と立証できる
3. 金額が**社会通念上妥当な範囲**である

契約書に「クリーニング費用は実費負担」としか書かれていない場合、業者の言い値で請求されても**妥当性を争う余地**があります。見積もりの内訳を必ず請求しましょう。

## 退去立会いのコツ｜その場で「サインしない」

退去費用トラブルの**8割は立会いの場で発生**します。ここでの対応が敷金返還額を左右します。

### 立会いで絶対やるべきこと

1. **立会い前に部屋の全箇所を写真・動画で撮影**（日付入り）
2. 指摘された汚損箇所は**一緒に確認し、写真で記録**
3. **「原状回復確認書」には即サインしない**
   - 「確認書にサインすると追加請求を受け入れたことになる」と主張される
   - 「いったん持ち帰って家族と確認してから返送します」でOK
4. 見積書は**項目・単価・数量・理由**まで明記させる
5. 「経年劣化ですよね？」「ガイドラインではどう扱われますか？」と積極的に質問

### やってはいけないこと

- 口頭での「了解しました」（録音されているケースあり）
- 白紙の確認書にサインしてしまう
- 立会いに同席せず業者任せにする（これが最悪パターン）

## 過剰請求への対処法｜5段階の交渉ステップ

いざ高額な請求書が届いたら、次の順番で対応します。

### ステップ1｜見積書の詳細を要求する

「単価・数量・施工範囲・写真」を明示した見積書を要求します。これが出てこなければ、その時点で**交渉の主導権は借主**に移ります。

### ステップ2｜ガイドラインを根拠に個別項目を精査

ガイドラインの表を横に置きながら、項目ごとに「経年劣化か故意・過失か」を仕分けます。経年劣化部分は**減額交渉**または**全額拒否**できます。

### ステップ3｜書面で減額を申し入れる

メールまたは内容証明郵便で、**「この項目はガイドラインに照らすと貸主負担と考えます。減額をお願いします」** と書面で送ります。書面化することで、後日訴訟になった場合の重要な証拠になります。

### ステップ4｜消費生活センター（消費者ホットライン188）に相談

減額交渉が進まない場合、**局番なし「188（いやや）」** で消費生活センターにつながります。専門家の助言や、業者への斡旋（あっせん）も依頼できます。**無料**で使えるため、活用しない手はありません。

### ステップ5｜少額訴訟を検討

請求額が**60万円以下**なら、簡易裁判所の「少額訴訟」が使えます。印紙代数千円＋1日で判決、という比較的手軽な制度です。ガイドラインに沿った主張をすれば、借主が勝訴するケースが多いのが実情です。

## 退去費用を減らすための「入居時からの備え」

退去時の交渉で優位に立つには、**入居時から証拠を残しておく**のが最善策です。引越し前後のやるべきことは「[引越し前後の手続きチェックリスト](/articles/moving-checklist)」でも解説していますが、ここでは退去費用対策にしぼってポイントを紹介します。

### 入居時にやるべきこと

1. **入居前・入居直後の写真を全部屋・全壁面・全床撮影**
2. 既存の傷・汚れを「入居時チェックリスト」に書き込んで大家さん・管理会社と共有
3. 契約書の**特約条項**をコピーして保管
4. 敷金の預託金額・振込先の記録

### 入居中にやるべきこと

1. 結露が出たらこまめに拭く（カビは借主負担リスク）
2. タバコは必ず換気扇の下 or ベランダ
3. ペット・DIYは契約を確認してから
4. 水漏れ・設備故障は**即管理会社に連絡**（放置は善管注意義務違反）

## ペットを飼っている場合の退去費用

ペット可物件の退去費用は、通常より高くなる傾向があります。

- 壁・柱の引っかき傷：借主負担（ほぼ確定）
- ペット臭：専門消臭で2〜5万円
- フローリングの爪痕：部分補修5〜20万円、重度なら張替え

敷金が2ヶ月分預かられているケースが多いですが、それでも足りずに追加請求されるケースもあります。入居時から**爪とぎ対策・床保護マット**を敷いておくのが最も効果的な対策です。

## FAQ（よくある質問）

### Q1. 退去費用の平均はいくら？

ワンルームで**3〜10万円**、2LDKで**5〜20万円**、3LDK以上で**10〜30万円**が目安です。過剰請求を受けている場合はこれより高額になりますが、ガイドラインを根拠に交渉すれば半額以下に減額できるケースも多いです。

### Q2. 敷金を返してもらえないときはどうすればいい？

まずは**明細書の提出を請求**します。明細を拒否する業者は要注意。請求根拠が不明なまま敷金を返さないのは**法的に無効**とされる判例もあります。消費者ホットライン188に相談するのが次のステップです。

### Q3. 「ハウスクリーニング特約」は絶対払わなきゃダメ？

契約書に**具体的な金額と項目**が明記されていれば、原則有効です。ただし**相場から著しく高額**（例：1Kで10万円など）なら、減額交渉の余地があります。内訳を請求し、社会通念上妥当か確認しましょう。

### Q4. タバコを吸っていた場合、全面張替えを請求される？

ヤニ汚れは借主負担ですが、**耐用年数を経過したクロスは残存価値ほぼゼロ**のため、新品張替え分の全額を請求されるのは不当です。6年以上住んでいた場合は「減価償却後の残存価値分のみ負担」が正しい計算です。

### Q5. 壁に画鋲の穴はOK？ ネジ穴はNG？

- **画鋲・ピンの小さな穴**：通常損耗扱いで**貸主負担**
- **クギ・ネジの穴で下地ボードまで達する**：借主負担

ただし下地を交換するほどの深い穴が多数あれば、**壁紙＋下地補修**で数万円請求されることもあります。

### Q6. 退去立会い当日、サインを求められたらどうする？

**「内容を持ち帰って確認したうえで返送します」** と伝え、その場でのサインは避けましょう。脅しのような言い方をされても、サインしない権利は借主にあります。不安なら立会いに第三者（家族・友人）を同席させるのも有効です。

### Q7. 契約書を失くしたらどうすれば？

管理会社にコピーを請求できます（手数料数百円〜数千円）。特約の内容確認や交渉には契約書が不可欠なので、引越し前に**必ず再入手**してください。

### Q8. 弁護士に相談するタイミングは？

請求額が**20万円以上**、かつ交渉が平行線になった時点で検討します。自治体の**法テラス（無料法律相談）** や、弁護士会の**30分5,000円程度の初回相談**を活用すれば、費用対効果よく動けます。

## まとめ｜退去費用は「知識」で半額にできる

退去費用トラブルは、知識格差が生んでいる側面が大きいと感じます。ガイドラインを読んでいない借主と、毎日のように交渉している業者では、そもそもスタート地点が違うからです。

しかし、この記事で紹介した**5つのポイント**を押さえるだけで、交渉力は一気に上がります。

1. 原状回復の基本は「故意・過失分のみ借主負担」
2. 経年劣化・通常損耗は貸主負担（クロス・畳・設備は6年で残存価値ほぼゼロ）
3. 特約は有効だが、金額の妥当性は争える
4. 立会いでその場サインはしない
5. 消費者ホットライン188・少額訴訟という公的な救済策がある

退去費用で損しないことは、次の引越しの軍資金を守ることと同じです。あわせて「[引越し費用の平均相場](/articles/moving-cost-average)」「[引越しを安くする裏ワザ](/articles/cheapest-moving-tips)」もご覧いただき、**退去時と引越し時の両面から**支出を最適化してください。

正しい知識は最強の盾です。今回の引越しがスムーズに終わり、敷金もしっかり戻ってきますように。
